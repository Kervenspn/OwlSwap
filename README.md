# OwlSwap
CSC 330 Final Project OwlSwap
BookSwap is a simple web app for SCSU students to sign up with an SCSU email, log in, create and manage book listings, browse others’ books, and message students to arrange swaps. Key features: SCSU-only registration, secure authentication, CRUD listings (title, author, edition, condition, photos, location), search and filters, in-app messaging, swap requests/accept/complete, admin moderation, and soft-delete/audit logging.
